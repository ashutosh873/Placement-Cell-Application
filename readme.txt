Steps to setup the project - 

1. Install node.js, mongoDB, mongoDB compass and IDE (eg. VS Code).

2. Add nodejs path to environment variables

3. Install the node modules/ dependencies as mentioned in package.json using node package manager (npm). 
Command - npm i <dependency_name>

4. Provide the environment variables - APP_PORT, MongoDBURL, JWT_SECRET_KEY

5. Start the server using the following command - node server.js

6. Application will be accessible on following URL - http://localhost:3000/employee/signup
